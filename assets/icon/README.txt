Place your app icon PNG file here and name it: app_icon.png

Requirements:
- Format: PNG
- Recommended size: 1024x1024 pixels (square)
- The image should have a transparent background or solid background
- Once the file is placed here, run: flutter pub get && flutter pub run flutter_launcher_icons

The icon will be automatically generated for:
- Android (all densities)
- iOS (all sizes)
- Web
- Windows
- macOS
